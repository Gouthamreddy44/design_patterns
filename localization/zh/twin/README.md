## 目的
双胞胎模式是一种设计模式，它提供了一个标准的解决方案来模拟多个
java中的继承

## 类图
![alt text](./etc/twin.png "Twin")

## 适用性
使用 Twin 成语时

* 在不支持此功能的语言中模拟多重继承。
* 避免多重继承的某些问题，例如名称冲突。

## 鸣谢

* [Twin – A Design Pattern for Modeling Multiple Inheritance](http://www.ssw.uni-linz.ac.at/Research/Papers/Moe99/Paper.pdf)
