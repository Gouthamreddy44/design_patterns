## 目的
将用户界面分为三个相互关联的组件：
模型、视图和控制器。让模型管理数据，视图
显示数据和控制器调解更新数据和重绘
展示。
## 类图
![alt text](./etc/model-view-controller.png "Model-View-Controller")

## 适用性
在以下情况下使用模型-视图-控制器模式

* 您希望将域数据与其用户界面表示清楚地分开
